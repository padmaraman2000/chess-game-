## 🕹️ How to Play Chess

| Piece | Movement |
|---|---|
| ♙ Pawn | Forward 1 (or 2 on first move), captures diagonally |
| ♖ Rook | Any number of squares horizontally or vertically |
| ♘ Knight | L-shape (2+1 squares), can jump over pieces |
| ♗ Bishop | Any number of squares diagonally |
| ♛ Queen | Any direction, any number of squares |
| ♔ King | One square in any direction |

**Special Moves:**
- **Castling** — King moves 2 squares toward Rook
- **En Passant** — Pawn captures passing pawn
- **Promotion** — Pawn reaching last rank becomes Queen/Rook/Bishop/Knight

**Win by:** Checkmating the opponent's King!
